# andignika.github.io

Personal academic website for **Andi Nika**.

This repository contains the source code for my personal homepage, hosted via **GitHub Pages** at:

**[https://andignika.github.io](https://andignika.github.io)**

## Overview

The website is a lightweight personal academic site built for clarity and ease of maintenance. It includes:

- a short biography
- research interests
- links to my CV, Google Scholar, and LinkedIn
- a publications and preprints section
- a responsive single-page layout

The site is designed to stay simple, fast, and easy to update.

## Tech stack

- **HTML**
- **CSS**
- **Jekyll** (via GitHub Pages)
- **YAML data files** for publication entries

No frontend framework or build step is required.
